### Movies Api oefening
Werk de startsituatie verder uit en zorg voor inderstaande functionaliteiten:
- Maak een MovieCardComponent die de basis info van een movie toont(titel en image).
- Maak een MovieListComponent die een lijst van MovieCards toont. gebruik hiervoor de MovieCard component.
- Maak een Movies page aan die de MovieList component zal aanroepen.
- Gebruik de aanwezige IApiService in de Movies page om de movies op te halen.
  - Maak gebruik van de aanwezige MovieModel en resultModel klassen om de data te displayen.
- Maak ook gebruik van de LoadingComponent om een loading animatie te tonen tijdens het laden.
- **Extra:** Zorg voor een Detail knop op de MovieCard component die, wanneer aangeklikt, de details van een movie laadt in een nieuwe MovieDetailComponent.
  - Maak hiervoor gebruik van het **Callback** mechanisme.
